## Install

```sh
$ npm i vscode-highlight-tagged-templates
```

## Usage

```js
import vscodeHighlightTaggedTemplates from 'vscode-highlight-tagged-templates'

console.log(vscodeHighlightTaggedTemplates())
//=> Hello World!
```
